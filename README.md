# E-Commerce Sales Analytics Dashboard

An interactive and professional E-Commerce Analytics Dashboard built using Python, Streamlit, Pandas, and Plotly to analyze sales performance, customer behavior, product trends, and payment insights.

---

#  Features

 Interactive Dashboard  
 Sales Analytics  
 Product Performance Analysis  
 Customer Insights  
 Payment Method Analysis  
 City-wise Sales Distribution  
 Monthly Sales Trends  
 KPI Metrics  
 Download Filtered Data  
 Dynamic Sidebar Filters  
 Responsive Visualization  

---

# Dashboard Preview

The dashboard provides:

- Total Revenue
- Total Orders
- Average Ratings
- Average Order Value
- Product Category Analysis
- Monthly Sales Trends
- Top Selling Products
- Payment Method Insights
- City-wise Sales Distribution
